# Reviewer web application

This workspace will contain the React/Vite TypeScript requester and reviewer experience. It owns presentation, browser-side state, accessibility behavior, and generated API clients. It does not calculate risk tiers, embed AWS credentials, or call ServiceNow directly.

The first implementation change must add a locked package manifest plus deterministic install, lint, type-check, unit-test, build, and accessibility-check commands and compose them into the root verification workflow.
